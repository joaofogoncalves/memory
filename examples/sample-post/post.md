# January 15, 2024

This is an example of what an archived LinkedIn post looks like.

The archiver preserves your original text, downloads any attached images or
videos into the media/ subdirectory, and generates this clean markdown file
with YAML frontmatter.

Posts are organized by date: posts/YYYY/MM/post-slug/post.md

Each post directory can also contain a media/ folder:
  posts/2024/01/this-is-an-example/
    post.md
    media/
      image-1.jpg
