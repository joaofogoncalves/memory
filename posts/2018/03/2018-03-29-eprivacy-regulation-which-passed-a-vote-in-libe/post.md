# March 29, 2018

ePrivacy Regulation which passed a vote in LIBE yesterday (and is expected to pass a vote across the entire EU Parliament next week) will impose a 4% penalty of the previous year's revenues (same as GDPR) for ANY and ALL software which permits communication that is in violation of the Regulation.

That includes: web sites, apps, operating systems (all platforms), IoT stacks - ALL SOFTWARE

So without wanting to make anyone too worried - I would advise developers to start taking privacy much more seriously than they currently do, because this law is now 2/3rds of the way through to adoption and that means when it comes into effect (probably summer 2019) you need to have your house in order.

And no, this is not a marketing post - it is a warning :)
