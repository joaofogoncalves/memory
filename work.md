# Work

Selected things I've built and shipped.

## Quorum

A chat-native agent that helps a team converge on what to build. It lives in the group chat, runs ideas through Ideation, Validation, and Planning with backflow when constraints change, and scores them against the team's real skills. Built end to end for Agents Day 2026.

Stack: Cloudflare Workers, Agents SDK, Workers AI (Llama 3.3 70B), TypeScript, Telegram
Link: github.com/joaofogoncalves/agents_day_challenge

## Bridgeport

An open-source deployment-management tool, shipped to v2.0.1.

Stack: TypeScript, Apache-2.0
Link: github.com/bridgeinpt/bridgeport

## Prospect

A full-stack TypeScript app that captures incoming project requests and triages them with AI: a plain-language summary, tags, and a risk checklist, using OpenAI structured outputs.

Stack: TypeScript, React, Fastify, Prisma, OpenAI structured outputs
Link: github.com/joaofogoncalves/prospect

## This site

Stack: Python, vanilla HTML, CSS, JavaScript
Link: github.com/joaofogoncalves/memory
